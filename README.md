# 洗衣房管理系统 java195

## 项目概述

洗衣房管理系统是基于`SpringBoot`和`Vue`的前后端分离架构，旨在提供高效的洗衣服务管理。本系统支持多平台运行，具备完善的角色权限管理和丰富的功能模块。

## 技术栈

- 后端：Java、SpringBoot、MyBatis、MySQL
- 前端：Vue.js
- 开发工具：IDEA、JDK1.8、Maven、node.js

## 系统角色

- 管理员
- 顾客

## 运行环境

- 系统环境：Windows/Mac

## 功能模块

### 管理员功能

- 个人中心
- 顾客管理
- 店家管理
- 店铺信息管理
- 衣物类型管理
- 洗衣信息管理
- 订单信息管理
- 订单进度管理
- 交流区管理
- 系统管理

### 店家功能

- 个人中心
- 店铺信息管理
- 衣物类型管理
- 洗衣信息管理
- 订单信息管理
- 订单进度管理

### 顾客功能

- 查看店铺信息
- 在交流区发言
- 个人中心
- 查看洗衣信息
- 订单信息管理
- 订单进度管理

## 核心亮点

- 前后端分离架构，易于管理和扩展
- 支持跨平台运行，满足不同操作系统的需求
- 详细的权限控制，确保系统安全稳定
- 功能模块丰富，涵盖洗衣房管理的主要需求

1. 准备后端运行环境，安装Java、MySQL等
2. 克隆项目至本地
3. 编译并运行后端服务
4. 前端项目依赖安装
5. 前端项目打包并部署
6. 根据实际需求配置系统参数
7. 启动系统，进行测试和运行
